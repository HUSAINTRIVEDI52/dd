# TaskFlow

A refined, minimal todo application designed for focus and clarity.

## Features

- **Minimalist Design**: Clean UI with a focus on typography and space.
- **Persistence**: Automatically saves your tasks to local storage.
- **Animations**: Fluid transitions powered by `motion`.
- **Filtering**: Easily view all, active, or completed tasks.
- **Cleanup**: One-click removal of completed items.

## Tech Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Motion** (Animations)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
